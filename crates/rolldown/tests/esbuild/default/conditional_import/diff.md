# Reason
1. esbuild will wrap `Promise.resolve().then() for original specifier`
# Diff
## /out/a.js
### esbuild
```js
// import.js
var require_import = __commonJS({
  "import.js"(exports) {
    exports.foo = 213;
  }
});

// a.js
x ? import("a") : y ? Promise.resolve().then(() => __toESM(require_import())) : import("c");
```
### rolldown
```js

//#region a.js
x ? import("a") : y ? import("./import.js") : import("c");

//#endregion
```
